Planos grátis e pago.

Nosso plano gratuito são os softwares Open Source hospedados em nosso GitHub e servidor no Discord com notícias e análises dos mercados Forex e de Cripomoedas.

E nosso plano pago são as licenças dos nossos indicadores na TradingView, para receber o acesso é necessário ser assinante DCA.

Logo após o pagamento você será redirecionado para criar uma conta em nosso site e receberá acesso a nossa Área de Membros onde oferecemos não obrigatórios mas essenciais cursos para Traders iniciantes e a nosso canal fechado no Discord apenas para membros.

Em nossa pagina para membros está hospedado os links para os indicadores tradingview, onde será necessário solicitar acesso exclusivo para membros, junto com os resultados das nossas Análises Preditivas.

Se você deseja usar um de nossos Softwares/Algoritmos localmente é necessario entender o básico de programação em Python e conhecimento em dados.

Nossos algoritmos de aprendizado de máquina foram escritos para rodar localmente na sua máquina sem a necessidade de informações da sua conta, sem APIs. É apenas necessário instalar algumas dependências para que funcione corretamente, por exemplo, TALib para plotar indicadores, Pandas para manipulação de dados, scikit-learn para algoritmo Regressão Linear, XGBoost para algoritmo de árvore de decisões, entre outros Frameworks de aprendizado de máquina.

Na Área de Membros DCA você vai encontrar artigos exclusivos para membros, os links para acessar nossos indicadores na TradingView, artigos exclusivos para membros e o link para nosso servidor oficial para Membros no Discord, onde você vai ter acesso a nossas análises exclusivas.

Para usar nossos indicadores é necessário ter conhecimento básico de Trading e a plataforma TradingView. Se você não sabe como fazer uma operação não se preocupe que enviaremos um PDF que ensina o básico para fazer uma analise gráfica e como entrar nesse mercado.

DCA permite usar o poder dos dados históricos do mercado para detectar tendências e estimar resultados em potenciais.

Os indicadores que desenvolvemos podem encontrar um desvio entre os valores "passados" e "futuros" do mercado financeiro. São utilizados para prever os movimentos de preços nos mercados financeiros. Não apenas geramos, mas também analisamos esses indicadores.

  • - Esteja presente na bolsa de valores, Forex e Criptomoedas!
  • - Seja antecipado para aprender sobre grandes mudanças no mercado e obter vantagem sobre sua concorrência!
  • - Reduza seus riscos - esteja preparado com bastante antecedência!

Oferecemos indicadores grátis na TradingView que podem ser acessados em nosso perfil na plataforma e indicadores exclusivos para membros, que são liberados para membros após o segundo mês de assinatura.
