Documentação

Nossas estratégias de negociação para Criptomoedas são scripts desenvolvidos na TradingView, a linguagem de programação que usamos para escrever nossos scripts é a PINE SCRIPT, uma linguagem de programação desenvolvida pela TradingView, a plataforma que vamos usar para as estratégias.

Também oferecemos algoritmos Machine Learning gratuitos [Open Source] escritos na linguagem de programação Python, você pode acessar ou baixar eles em nosso repositório na GitHub e tirar suas dúvidas no servidor Discord .

Para usar nossas estratégias tem um custo .

Logo após o pagamento você será redirecionado para o nosso Discord, lá temos membros com planos grátis e planos pagos, nossos membros pagos tem acesso a canais exclusivos dentro do nosso servidor.

Veja em nossa Documentação como usar.

Se você deseja usar um de nossos Softwares/Algoritmos localmente é necessario entender o básico de programação em Python e conhecimento em dados.

Nossos algoritmos de aprendizado de máquina foram escritos para rodar localmente na sua máquina sem a necessidade de informações da sua conta, sem APIs. É apenas necessário instalar algumas dependências para que funcione corretamente, por exemplo, TALib para plotar indicadores, Pandas para manipulação de dados, scikit-learn para algoritmo Regressão Linear, XGBoost para algoritmo de árvore de decisões, entre outros Frameworks de aprendizado de máquina.

DCA permite usar o poder dos dados históricos do mercado para detectar tendências e estimar resultados em potenciais.

Os indicadores que desenvolvemos podem encontrar um desvio entre os valores "passados" e "futuros" do mercado financeiro. São utilizados para prever os movimentos de preços nos mercados financeiros. Não apenas geramos, mas também analisamos esses indicadores.

  • - Seja antecipado para aprender sobre grandes mudanças no mercado e obter vantagem sobre sua concorrência!
  • - Reduza seus riscos - esteja preparado com bastante antecedência!
